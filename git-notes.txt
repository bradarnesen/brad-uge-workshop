git init initializes a new empty git repo
git log - shows history of commits
git status - shows you status of current git repo and suggest things to do
git add - stages changes for commits, or adds them to the index
git branch - creates a new branch
git commit - takes a snapshot of the current state of the code
git checkout - switches branches
git config - set configurations values of git
git remote - connects to github
git push - pushes to github
